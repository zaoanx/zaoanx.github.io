OpenTokRTC部署

发表于 2016-03-12   |   分类于 技术   |     |    阅读

OpenTokRTC简介：

OpenTokRTC是由Tokbox公司开源一个基于WebRTC的音视频通讯Web程序，拥有简单的视频会议功能。

项目地址：

OpenTokRTC

部署步骤（Ubuntu 14.04.2 LTS）：

  1. 在服务器上安装web容器
  2. 安装node.js
  3. 开始部署
    1. 检出工程 repo
    2. 在TokBox上注册获取API Key和Secret
    3. 将config.js中的Your opentok key和Your opentok secret替换成你自己的API Key和Secret
    4. 运行npm install来安装依赖的包
    5. 运行服务node app.js
  4. HTTPS问题
    如果服务器没有添加HTTPS安全验证，那么该Web程序将没有权限访问麦克风和摄像头硬件，所以我们应该给服务器添加HTTPS安全验证。或者出于测试的目的，我们可以先在Chrome浏览器中加入这一段--unsafely-treat-insecure-origin-as-secure="http://YOUR HOST:PORT" --user-data-dir=/test/only/profile/dir。
  5. 打开浏览器输入http://YOUR HOST:PORT
